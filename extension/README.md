# 핫딜우주 수집기

크롬에서 직접 쿠팡 특가 페이지를 열고, 버튼으로 상품을 수집해서 핫딜우주 사이트에 전송하는 개인용 확장프로그램입니다.

## 설치

1. Chrome 주소창에 `chrome://extensions` 입력
2. 오른쪽 위 `개발자 모드` 켜기
3. `압축해제된 확장 프로그램을 로드합니다` 클릭
4. 이 폴더 선택: `extension`

## 사용

1. 확장프로그램 아이콘 클릭
2. 사이트 주소: `https://www.hotdealspace.com`
3. 관리자 토큰: Vercel에 넣은 `ADMIN_UPDATE_TOKEN`
4. `설정 저장`
5. `골드박스 열기` 또는 `판매자특가 열기`
6. 쿠팡 페이지가 열린 상태에서 `현재 탭 수집`
7. 수집 개수 확인 후 `사이트로 전송`

전송이 성공하면 GitHub 데이터 파일이 업데이트되고, Vercel이 자동 배포합니다.
